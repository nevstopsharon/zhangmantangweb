# 02 基础规范

## 1. 颜色
- 页面背景：`#FFFFFF`
- 卡片背景：`#FFFFFF`
- 强调色：`#C8A060`
- 主文字：`#1A1A1A`
- 次文字：`#6A6868`
- 辅助文字：`#A8A4A0`
- 弱边框：`#D6D2CE`
- 首页首屏底色兜底：`#D4CEC6`

## 2. 字体与字号
- 正文字体：`Noto Sans SC, sans-serif`
- 标题字体：`Noto Serif SC, serif`
- 字重主值：`300`
- 常规补充值：`400`
- 基础字号：`16px`
- 正文行高：`1.6`
- 一级标题：`48px / 1.15`
- 二级标题：`24px / 1.3`

## 3. 间距
- `s = 16px`
- `m = 40px`
- `l = 90px`
- `xl = 120px`

说明：
- 页面主要横向留白使用 `l`
- 章节上下留白使用 `xl`
- 卡片和组件内部常用 `s` 与 `m`

## 4. 容器与边界
- 站点最大宽度：`1440px`
- 页面不再依赖左右限制线定义整体边界
- 页面秩序由容器宽度、区块留白、标题层级和局部分隔线建立
- 主边框：`1px solid #1A1A1A`
- 弱边框：`1px solid #D6D2CE`

## 5. 阴影与动效
- 卡片阴影：`0 10px 24px rgba(26, 26, 26, 0.1)`
- 详情页媒体导航按钮默认阴影：`0 10px 24px rgba(26, 26, 26, 0.14)`
- 详情页媒体导航按钮悬停阴影：`0 14px 28px rgba(26, 26, 26, 0.18)`
- 标准过渡：`0.2s`
- 卡片悬停上移：`-4px`
- 详情页媒体导航按钮悬停上移：`-2px`

## 6. 组件级基础结论
- 搜索推荐面板与筛选下拉面板都使用浮层逻辑，必须高于普通页面内容层级。
- 详情页的大图区优先保护作品完整显示，左右按钮不得压住主图内容。
- 多图详情页中的主图、缩略图、左右切换按钮共用同一当前图片状态。
- 展览与新闻的年份轨道属于显式导航，而不是辅助刻度；当前态必须一眼可识别。
- 联系模块存在两个变体：首页完整版和内页紧凑版。

## 7. 当前排版结论
- 首页、列表页和 about 页依赖“大标题 + 大留白 + 局部分隔线”的秩序感。
- 二级信息统一降为中灰色，不与标题抢层级。
- 站点目前不使用粗体标题作为主层级工具，主要靠字号、留白和位置关系区分。
