# 04 布局与响应式

## 1. 总体容器
- 页面主体最大宽度：`1440px`
- 页面居中
- 左右保留弱边框

## 2. 核心布局模式
### 首页
- 首屏：整屏视觉图
- 精选作品：`1 + 3` 排布
- 展览：单条重点展览
- 联系模块：全站统一底部区块

### 列表页
- 作品页：统一三列卡片
- 展览页：左侧年份轨道，右侧纵向卡片流
- 新闻页：左侧年份轨道，右侧纵向卡片流

### 详情页
- 左侧年份轨道，右侧主内容
- 图片先展示，信息区紧随其后
- 新闻详情与展览详情采用相近信息层级

## 3. 响应式断点
### 小于等于 `1100px`
- 导航改为单列堆叠
- 首屏改为自动高度
- 年份轨道改为横向
- 详情区和联系模块改为单列
- 三列网格改为两列

### 小于等于 `760px`
- 横向留白缩小为 `24px`
- `h1` 缩小为 `38px`
- `h2` 缩小为 `22px`
- 三列网格改为单列
- 搜索框宽度占满工具区

## 4. 当前布局判断
当前官网的布局重点不是“信息量最大化”，而是“观看顺序清晰”：
- 先看品牌和主图
- 再看代表作
- 再进入列表浏览
- 最后在每页底部承接联系
