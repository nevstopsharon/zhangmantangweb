# 06 页面模式

## 1. 首页
结构：
1. 品牌导航
2. 主视觉封面
3. 精选作品 `3 x 2`
4. 近期展览 `1`
5. 首页完整版联系模块

## 2. 作品页
结构：
1. 页面主标题
2. 筛选器
3. 三列作品卡片网格

说明：
- 当前只保留一个主标题，不再显示重复小标题

## 3. 展览页
结构：
1. 页面主标题
2. 左侧年份轨道
3. 右侧展览卡片流

说明：
- 仅保留一个主标题，不重复显示较小字号标题

## 4. 新闻页
结构：
1. 页面主标题
2. 左侧年份轨道
3. 右侧新闻卡片流

说明：
- 卡片内部不重复显示底部灰色日期

## 5. 关于页
结构：
1. 主标题
2. 左侧一主一辅人物照片组合
3. 右侧人物介绍与单列展签式时间轴
4. 全站联系模块

## 6. 详情页
### 作品详情
- 返回入口
- 标题
- 主图
- 图框外侧左右切换按钮（仅多图时出现）
- 缩略图
- 左正文 / 右信息
- 内页紧凑联系模块
- 不再出现“全部 / 项目 / 材质”筛选器

### 展览详情
- 返回入口
- 左侧年份轨道
- 标题
- 主图
- 图框外侧左右切换按钮（仅多图时出现）
- 缩略图
- 图下紧凑信息区
- 内页紧凑联系模块
- 点击年份轨道中的年份点：返回展览总页并滚动到对应年份分组

### 新闻详情
- 返回入口
- 左侧年份轨道
- 标题
- 主图
- 图框外侧左右切换按钮（仅多图时出现）
- 缩略图
- 左正文 / 右信息
- 可选链接区：相关报道、完整视频
- 内页紧凑联系模块
- 点击年份轨道中的年份点：返回新闻总页并滚动到对应年份分组
