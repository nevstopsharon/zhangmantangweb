# D 盘文件夹架构设计

日期：2026-04-07

## 目标

把 `D:\` 设计成一套浅层、直观、可长期维护的个人文件结构。

核心约束：

- 根目录只保留少量高辨识度入口。
- 文件夹不做过度层叠。
- 能从文件名区分的信息，不再额外加一层文件夹。
- 不允许为了单个文件单独建文件夹。
- 不触碰系统目录、游戏平台目录和明显依赖固定路径的软件目录。

## 设计原则

按第一性原理，文件系统的效率来自三件事：

1. 找文件时先按用途判断，而不是先猜来源。
2. 每个文件只应有一个稳定归属，避免重复放置。
3. 路径承担“缩小范围”的职责，文件名承担“说明具体内容”的职责。

因此本方案采用：

- 根目录按生活领域分类。
- 默认只到两层。
- 第二层只有在它能明显减少搜索范围时才保留。

## 最终推荐架构

```text
D:\
├─ 00_下载待整理
├─ 01_证件申请
├─ 02_学习作品证书
├─ 03_工作业务
├─ 04_家庭亲友
├─ 05_照片视频
├─ 06_兴趣收藏
├─ 07_字体安装包
└─ 08_私密资料
```

## 每个入口的职责

### `00_下载待整理`

只承担导入和待分拣职责，不做长期存放。

适合内容：

- 浏览器下载
- 网盘下载
- 迅雷下载
- 临时中转

### `01_证件申请`

放所有与身份、证明、求职、学位申请、签证申请相关的材料。

保留当前这一级子目录是合理的，因为每类材料数量多、时间跨度长：

- `求职材料`
- `学位申请`
- `签证与出入境`
- `认证与证明`
- `通用材料`

### `02_学习作品证书`

放个人学习、作品、课程、培训证书、练习项目。

适合内容：

- 用户体验设计作品集
- 培训证书
- 数据可视化练习
- 课程与术语
- 电子书和自学资料
- 作业修改记录

### `03_工作业务`

放实习资料和家族企业相关资料。

适合内容：

- 实习公司资料
- 家族企业资料
- 发票和表格
- 活动方案、设计文件、名单

### `04_家庭亲友`

按人和事件存放家庭、亲友相关内容。

适合内容：

- 陈惠玲照片
- 陈惠玲聚会视频
- 家庭照片

### `05_照片视频`

放你自己的照片、视频、剪辑素材、游戏视频。

适合内容：

- 相机照片
- 大疆视频
- vlog 成片和脚本
- 剪映素材
- 游戏录屏

### `06_兴趣收藏`

放与兴趣、爱好、收藏相关的资料。

适合内容：

- 胡歌照片收藏
- 摩尔庄园周边图片
- 游戏记录表

### `07_字体安装包`

放字体和安装包。

适合内容：

- 字体目录
- 软件安装包

### `08_私密资料`

放不希望和普通资料混放的敏感内容。

适合内容：

- 特定人物资料
- 私人文稿

## 不移动的目录

以下目录不纳入人工整理范围，保持原位：

- `Program Files`
- `Program Files (x86)`
- `WindowsApps`
- `WpSystem`
- `Recovery`
- `System Volume Information`
- `$RECYCLE.BIN`
- `SteamLibrary`
- `Epic Games`
- `Riot Games`
- `XboxGames`
- `Xbox_GameFiles`
- `RiotGames_GameFiles`
- `Delta Force`
- 其他明显属于游戏安装目录的文件夹

## 命名规则

### 文件夹命名

只用“类别_对象”或“对象_内容”。

示例：

- `实习公司_蜂鸟小课`
- `陈惠玲_聚会视频`
- `个人视频_大疆Pocket3`
- `胡歌_照片收藏`

### 文件命名

统一规则：

`YYYY-MM-DD_原文件名`

说明：

- 只补日期前缀。
- 原文件名主体不改。
- 已经自带日期、时间戳、设备编号、相机编号的文件保持不动。

## 暂缓处理

以下项目目前不建议自动搬动，原因是归属不够单一或可能依赖路径：

- `D:\END`
- `D:\.GamingRoot`
- `D:\desktop.ini`
- `D:\Windows_Update_Cache`
- `D:\Fxhash_数字艺术品登陆验证`

## 推荐执行顺序

1. 先重命名当前顶层入口目录。
2. 再搬运下载类目录到 `00_下载待整理`。
3. 再处理 `01_证件申请` 和 `02_学习作品证书`。
4. 然后处理 `03_工作业务`、`04_家庭亲友`、`05_照片视频`。
5. 最后处理 `06_兴趣收藏`、`07_字体安装包`、`08_私密资料`。

这样做的原因是：

- 先处理高确定性材料，错误概率最低。
- 先稳定入口，再整理细项，后续不会反复移动。
- 把模糊项留到最后，避免在不确定的地方浪费时间。
