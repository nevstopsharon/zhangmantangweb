# 技术架构文档

## 系统架构
本项目采用前端单页应用架构，使用原生 JavaScript 实现路由和状态管理。

### 核心模块
1. **状态管理** (`state.js`)
   - 管理应用状态
   - 处理多语言切换
   - 维护过滤和搜索状态

2. **路由系统** (`router.js`)
   - 基于哈希的路由
   - 处理页面导航
   - 支持多语言路由

3. **数据加载** (`data-loader.js`)
   - 加载 JSON 数据
   - 处理数据缓存
   - 增强错误处理

4. **渲染系统** (`renderers.js`, `renderers-secondary.js`)
   - 渲染页面内容
   - 处理不同类型的页面

5. **组件系统** (`components.js`)
   - 提供通用组件
   - 处理模板渲染
   - 提供工具函数
   - 防御性编程

6. **主应用** (`main.js`)
   - 初始化应用
   - 绑定事件处理
   - 协调各个模块
   - 图片懒加载

## 数据流
1. 应用初始化时加载 JSON 数据
2. 用户交互触发状态变化
3. 状态变化触发页面重新渲染
4. 渲染系统根据当前状态生成 HTML
5. 页面更新，展示新内容

## 多语言支持
- 使用 JSON 存储多语言内容
- 通过状态管理切换语言
- 路由系统支持多语言 URL

## 性能优化
- 图片懒加载
- 资源预加载
- 关键 CSS 内联
- 缓存策略

## 安全性
- 输入验证和转义
- 安全头部设置
- 防御性编程

## 部署架构
- 静态文件部署
- 支持 GitHub Pages / Vercel
- 无需服务器端处理
